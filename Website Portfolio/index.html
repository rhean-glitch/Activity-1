About Me

Hi! I'm Renan Baguio, a 20-year-old second-year BSIT student at Cordova Public College, living in Catarman, Cordova, Cebu. I'm passionate about technology, especially web development, where I enjoy creating clean, responsive, and user-friendly websites.

My interest in websites started with curiosity and soon grew into a real passion. As I learned HTML, CSS, and JavaScript, I realized how much I enjoy turning ideas into working websites. Now, I'm exploring tools like React and Git to improve my skills even more.

I'm always excited to learn new things and take on challenges that help me grow. Whether it’s a school project or a personal task, I do my best to write clean code and build useful, well-designed websites.

As a student, I value learning, teamwork, and hands-on experience. I'm curious, hardworking, and ready to keep improving as I work toward a future in the IT industry.
